# DungeonFighter
DungeonFighter，APRG Mobile Game.<br>
MVC架构，对象缓冲池，小物件层消隐，手动遮挡剔除。<br>
常用设计模式应用（观察者、责任链等）<br>
with Unity5.3.6 + UGUI + EasyTouch + DOTween<br><br>
